# 单词记忆卡片应用

## 项目简介

基于Vue3开发的单词记忆卡片应用，采用简化的间隔重复算法，帮助用户高效记忆英语单词。应用支持单词的添加、删除和管理，带有流畅的卡片翻转动画，并提供可视化的学习进度跟踪。

## 技术栈

| 技术 | 版本 | 用途 |
|------|------|------|
| Vue | 3.5.24 | 前端框架 |
| Vite | 7.3.0 | 构建工具 |
| JavaScript | ES6+ | 编程语言 |
| CSS3 | - | 样式设计 |
| LocalStorage | - | 本地数据存储 |

## 核心功能

### 1. 记忆测试
- 带有流畅的3D卡片翻转动画
- 支持"记住了"和"没记住"两种状态
- 基于简化的间隔重复算法自动调整复习时间

### 2. 单词本
- 支持添加新单词
- 支持删除单词
- 显示单词状态和下次复习时间
- 按状态分类显示单词

### 3. 学习进度
- 可视化展示学习进度
- 显示总单词数、已复习单词数、待复习单词数
- 提供学习建议

## 安装和运行

### 环境要求
- Node.js >= 16.0.0
- npm >= 8.0.0

### 安装步骤

1. 克隆仓库
```bash
git clone https://github.com/Rinch9999/wordcard.git
cd wordcard
```

2. 安装依赖
```bash
npm install
```

3. 启动开发服务器
```bash
npm run dev
```

4. 访问应用
   - 打开浏览器访问：http://localhost:5173/

### 构建生产版本

```bash
npm run build
```

构建产物将生成在 `dist` 目录中。

### 本地预览生产版本

```bash
npm run preview
```

## 间隔重复算法说明

本应用采用简化的间隔重复算法，根据用户的学习情况自动调整单词的复习时间：

- **记住了**：下次复习时间 = 当前时间 + 原间隔时间 × 难度系数（默认2.5）
- **没记住**：下次复习时间 = 当前时间 + 1天（重置间隔）

## 数据存储

应用使用浏览器的LocalStorage进行数据存储，无需后端支持。所有单词数据都保存在浏览器本地，刷新页面或重新打开浏览器后数据不会丢失。
