# 单词记忆卡片应用

## 项目简介

基于Vue3开发的单词记忆卡片应用，采用简化的间隔重复算法，帮助用户高效记忆英语单词。应用支持单词的添加、删除和管理，带有流畅的卡片翻转动画，并提供可视化的学习进度跟踪。

## 核心功能

### ✅ 记忆测试
- 带有流畅的3D卡片翻转动画效果
- 支持"记住了"和"没记住"两种学习状态
- 基于简化的间隔重复算法自动调整复习时间
- 自动切换到下一个待学习单词

### ✅ 单词本管理
- 支持手动添加新单词
- 支持删除现有单词
- 显示单词状态（新单词/已复习/需复习）
- 显示下次复习时间
- 按状态分类展示单词

### ✅ 学习进度跟踪
- 可视化进度条展示总体学习进度
- 统计总单词数、已复习单词数、待复习单词数
- 提供个性化学习建议
- 30秒自动刷新进度数据

### ✅ 本地数据存储
- 使用浏览器LocalStorage持久化存储
- 无需后端服务器支持
- 数据自动保存，刷新页面不丢失

## 核心功能使用说明

### 1. 记忆测试

1. 进入**记忆测试**页面
2. 点击卡片查看单词释义
3. 根据记忆情况选择"记住了"或"没记住"
4. 系统自动调整下次复习时间并切换到下一个单词
5. 完成所有待复习单词后，显示学习完成提示

### 2. 单词本管理

1. 进入**单词本**页面
2. 在顶部输入框中输入单词和释义
3. 点击"添加单词"按钮保存
4. 单词会显示在下方列表中
5. 点击单词右侧的"删除"按钮可删除指定单词
6. 单词卡片显示状态标签和下次复习时间

### 3. 学习进度

1. 进入**学习进度**页面
2. 查看统计卡片：总单词数、已复习、待复习
3. 查看进度条：显示总体完成率
4. 查看学习建议：根据待复习单词数量给出建议

## 间隔重复算法说明

本项目采用简化的间隔重复算法，原理如下：

- **记住了**：下次复习时间 = 当前时间 + 原间隔时间 × 难度系数（默认2.5）
- **没记住**：下次复习时间 = 当前时间 + 1天（重置间隔）
- **初始间隔**：所有新单词初始间隔为1天
- **难度系数**：固定为2.5，简化实现

## 数据结构说明

每个单词包含以下字段：

| 字段名 | 类型 | 描述 |
|--------|------|------|
| id | Number | 唯一标识 |
| word | String | 单词 |
| meaning | String | 释义 |
| nextReview | String | 下次复习时间（ISO格式） |
| interval | Number | 复习间隔（天） |
| easiness | Number | 难度系数 |
| status | String | 状态：new/reviewed/failed |

## 常见问题（FAQ）

### Q: 如何重置所有数据？
A: 打开浏览器开发者工具（F12）→ Application → Local Storage → 删除 `words` 条目即可重置

### Q: 为什么有些单词没有出现在记忆测试中？
A: 只有下次复习时间小于等于当前时间的单词才会显示在记忆测试中

### Q: 支持导入外部单词表吗？
A: 当前版本暂不支持，后续可以考虑添加此功能

### Q: 可以在移动端使用吗？
A: 支持，应用采用响应式设计，适配不同屏幕尺寸

## 开发说明

### 核心代码文件

1. **App.vue**：主应用组件，导航和页面切换
2. **MemoryCard.vue**：记忆卡片组件，翻转动画和学习状态处理
3. **WordList.vue**：单词本组件，单词的添加和删除
4. **ProgressTracker.vue**：进度跟踪组件，统计和可视化
5. **wordService.js**：数据服务，本地存储和间隔重复算法

### 组件通信

- 组件通过调用 `wordService.js` 中的函数获取和更新数据
- 使用Vue3的响应式系统自动更新UI
- 无复杂的组件间通信，主要通过共享数据服务
